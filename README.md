# User API

A REST API built with Python, FastAPI and PostgreSQL. 
Containerized with Docker and deployed on AWS EC2.
Automatically deploys via GitHub Actions CI/CD pipeline.

## Features
- Full CRUD for users
- PostgreSQL database
- Dockerized with Docker Compose
- Automated deployment with GitHub Actions

## Tech Stack
- Python + FastAPI
- PostgreSQL
- Docker + Docker Compose
- AWS EC2
- GitHub Actions

## Live API
http://3.16.113.74:8000/docs

## How to Run Locally
1. Clone the repo
2. Run: docker compose up
3. Open: http://localhost:8000/docs
